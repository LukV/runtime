# Released

Narrative release history for Runtime. The repo's `CHANGELOG.md` (when it exists) is the semver shipping log — every conventional commit, every version bump. *This* file is the human-readable record of **what each release was for**.

Releases are named, not numbered, in the spirit of Vlaamse jogging tradition — local races have names, not version numbers. Each named release rolls up multiple shipped pitches.
